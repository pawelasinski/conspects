Пример правильного кода¶

Ваш код - текст, который много раз будет читаться вами и вашими коллегами. Таким образом, легче один раз написать код понятно, чем сто раз его перечитывать и, возможно, разбираться в нем с нуля :-)

Для того, чтобы ваш код выглядел как английский текст, нужно соблюдать ряд правил:

  • Называйте переменные осмысленно. Это очень сильно облегчит вам жизнь, поскольку вы сможете по названию функции понять, что она делает (не залезая в её код), а по названию переменной сможете сразу сказать, что в ней лежит.
  • Используйте комментарии. Комментарии начинаются с решетки (#). Это куски кода, которые не исполняются программой, но позволяют внести дополнительную информацию о коде. Комментирование кода также позволит легче ориентироваться в блоках самой программы :)
  • Используйте докстрингу. Докстринга (Docstring) - многострочный текст, заключенный в три кавычки (либо '''текст''', либо """текст"""). Этот текст используется для документирования, в частности, функций. В нем вы можете расписать что делает функция, что принимает на вход, что возвращает на выход и т.д.
  • Старайтесь соблюдать структуру. Это особенно критично, когда вы пользуетесь Jupyter'ом. В процессе работы часто приходиться "прыгать" по ячейкам, поэтому могут возникать моменты "в 20-ой ячейке создал функцию, которую применил в 3-ей". Когда вы снова вернетесь к коду "с чистого листа", что-нибудь может поломаться :)
  • Старайтесь соблюдать PEP 8. Его можно назвать руководством по написанию кода. PEP 8 позволяет унифицировать код, тем самым делая его более понятным. Ссылка на PEP 8 - https://www.python.org/dev/peps/pep-0008/

Помните, что программист тратит 90% времени на чтение кода :-)

# Создадим вспомогательную функцию

def print_lines_of_file(path_to_file, n_lines=None):
    """
    Данная вспомогательная функция принимает путь к файлу
        и выводит на печать заданное количество строк из файла.

    Параметры
    ---------
    path_to_file: путь к файлу
    n_lines: количество строк, которые нужно напечатать.
        Если этот параметр не указан, то на печать выведется весь текст построчно

    Функция возвращает None
    """
    with open(path_to_file) as check_format:

        if n_lines is None:
            # Выводим весь текст на печать
            for line in check_format:
                print(line.strip()) #.strip() убирает лишние пробелы в начале и конце строки

        else:
            # Выводим на печать n_lines строк
            for line, _ in zip(check_format, range(n_lines)):
                print(line.strip())

# Зададим функцию для расчета оценки

def calculate_grade(score):
    """
    Функция принимает балл студента (в диапазоне 0 - 100) и переводит его в оценку. 
    Оценка возвращается в виде строки

    score: балл студента (от 0 до 100)

    Функция возвращает str
    """
    if score >= 90:
        return "Отлично"
    elif score >= 75:
        return "Хорошо"
    elif score >= 60:
        return "Удовлетворительно"
    return "Незачёт"
